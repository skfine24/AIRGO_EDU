# 🛸 AIRGO EDU

## 🇺🇸 English
**Easy and safe for everyone! A customized drone education solution for students from elementary to university.**

AIRGO EDU is a professional educational drone that covers everything from basic drone principles to advanced Python coding. Available as both a DIY assembly kit and a finished product, it provides the optimal learning environment for any educational goal.

### ✨ Key Features
* **🛡️ Stable Flight Performance**: High-performance sensors for automatic altitude hold and stable hovering.
* **👁️ Optical Flow Sensor**: Precise position control using a downward sensor for drift-free indoor flight.
* **📷 WIFI Camera**: Supports real-time video streaming and recording.
* **🎮 2.4GHz Wireless Controller**: Interference-free, stable connection for immediate control.
* **🛠️ DIY Kit & Ready-to-Fly**: Choose between learning drone architecture with the assembly kit or flying immediately with the finished product.

### 💻 Coding & Software Education
* **App-based Block Coding**: Learn intuitive drone control algorithms via mobile app.
* **Entry**: Visual programming support optimized for educational environments.
* **Python**: Advanced drone control and data utilization through text-based coding.

### 🎓 Target Audience
* **Elementary School**: Basic flight control and logical thinking through block coding.
* **Middle & High School**: Understanding hardware structure and intermediate mission execution.
* **University**: Autonomous flight and application development using Python libraries.

## 🇰🇷 한국어
**누구나 쉽고 안전하게! 초등학생부터 대학생까지 함께하는 맞춤형 드론 교육 솔루션**

AIRGO EDU는 드론의 기초 원리부터 고급 파이썬 코딩까지 학습할 수 있는 교육 전문 드론입니다. 조립 DIY 키트와 완제품 두 가지 형태로 제공되어 사용자의 목적에 맞는 최적의 교육 환경을 지원합니다.

### ✨ 주요 특징
* 🛡️ 안정적인 비행 성능**: 고성능 센서를 통한 자동 고도 유지 및 안정적인 호버링 지원
* 👁️ 옵티컬 플로우 센서 (Optical Flow Sensor)**: 하단 센서를 이용한 정밀한 위치 제어로 실내에서도 흔들림 없는 비행 가능
* 📷 WIFI 카메라**: 실시간 영상 스트리밍 및 촬영 지원
* 🎮 2.4GHz 무선 조종기**: 간섭 없는 안정적인 연결로 즉각적인 컨트롤 가능
* 🛠️ 조립 DIY 키트 & 완제품**: 기체 구조를 배우는 조립형과 바로 비행하는 완제품 중 선택 가능

### 💻 코딩 및 소프트웨어 교육
* **블록 코딩 (App-based)**: 앱을 이용한 직관적인 드론 제어 알고리즘 학습
* **엔트리 (Entry)**: 국내 교육 현장에 최적화된 블록 기반 비주얼 프로그래밍 지원
* **파이썬 (Python)**: 텍스트 코딩을 통한 고등 교육 수준의 드론 제어 및 데이터 활용

### 🎓 교육 대상
* **초등학교**: 드론 조종의 기초와 논리적 사고력을 키우는 블록 코딩 수업
* **중·고등학교**: 드론 하드웨어 구조 이해 및 중급 수준의 미션 수행 교육
* **대학교**: 파이썬 라이브러리를 활용한 자율 주행 및 응용 애플리케이션 개발

## 🇯🇵 日本語
**誰でも簡単かつ安全に！小学生から大学生まで対応したカスタマイズドローン教育ソリューション**

AIRGO EDUは、ドローンの基礎原理から高度なPythonコーディングまで学習できる教育専門ドローンです。組み立てDIYキットと完成品の2つの形態で提供され、教育目的に合わせた最適な環境をサポートします。

### ✨ 主な特徴
* **🛡️ 安定した飛行性能**: 高性能センサーによる自動高度維持と安定したホバリング。
* **👁️ オプティカルフローセンサー**: 下部センサーによる精密な位置制御で、室内でもブレのない飛行が可能。
* **📷 WIFIカメラ**: リアルタイムの映像ストリーミングおよび撮影をサポート。
* **🎮 2.4GHz 無線プロポ**: 干渉のない安定した接続で即座にコントロール可能。
* **🛠️ 組み立てDIYキット＆完成品**: 機体構造を学ぶ組み立て型と、すぐに飛ばせる完成品から選択可能。

### 💻 コーディング & ソフトウェア教育
* **アプリブロックコーディング**: アプリを使用した直感的なドローン制御アルゴリズムの学習。
* **Entry**: 教育現場に最適化されたブロックベースのビジュアルプログラミング。
* **Python**: テキストコーディングによる高等教育レベルのドローン制御およびデータ活用。

### 🎓 教育対象
* **小学校**: ドローン操縦の基礎と論理的思考力を養うブロックコーディング授業。
* **中学校・高校**: ドローンのハードウェア構造の理解と中級レベルのミッション遂行教育。
* **大学**: Pythonライブラリを活用した自律走行および応用アプリケーション開発。
